# Incorrect parameter passed

## Summary
The report is about a bug in the Curve swap function, which can sometimes return more than the minimum value specified. The recommendation is to use the balance difference as a parameter to fix this issue.

## Details
##### Description
Curve swap can return more than `min_dy`:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/adapters/CurveV1.sol#L82
##### Recommendation
We recommend to use balance difference as a parameter for the function.
