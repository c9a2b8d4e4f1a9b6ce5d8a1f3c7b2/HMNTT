# Unable to remove pool or manager

## Summary
This bug report concerns a problem with the Gearbox Protocol's Contracts Register, which is a set of code used to manage the protocol's pools and managers. Specifically, the code does not include functions for removing pools and managers, making it difficult to update the protocol. The bug report recommends adding functions for removing pools and managers to the code to fix this issue.

## Details
##### Description
Pool or manager can't be removed:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/core/ContractsRegister.sol#L38
##### Recommendation
We recommend adding functions for removing pools and managers.
