# Index not checked

## Summary
This bug report is about an issue with the index 'i' in the Gearbox Protocol codebase. The index 'i' must be less than the variable 'N_COINS' in order for the code to function properly. The code can be found in two places in the Gearbox Protocol codebase: lines 40 and 58 of the 'CurveV1.sol' file. The recommendation is to add a check that 'i < N_COINS' in order to ensure the code is functioning correctly.

## Details
##### Description
Index `i` must be less than `N_COINS`:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/adapters/CurveV1.sol#L40
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/adapters/CurveV1.sol#L58
##### Recommendation
We recommend adding a check that `i < N_COINS`.
