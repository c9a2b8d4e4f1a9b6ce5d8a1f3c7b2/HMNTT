# Integer overflow/underflow

## Summary
The report highlights a bug that occurs in multiple sections of the Basilisk-node and HydraDX-math code. The bug is caused by not using safe math functions, which can lead to potential issues. The recommendation is to use safe functions like checked_add() and checked_sub() for math operations to avoid this bug.
