# View function will never return value.

## Summary
This bug report is about an error in the code of a contract called StakingB.sol. The error is on line 46 and it is related to a function called "stakesInfo". This function is supposed to return a value, but it is not working properly because it creates an array that is not correctly formatted. This means that the function is not able to iterate through the array and therefore cannot return the desired value. To fix this, the recommendation is to change the format of the array or to use a different method to iterate through it.

## Details
**Description**

StakingB.sol Line 46
“stakesInfo” will never return value because it creates an array [from; to) but tries to iterate
through [from; to].

**Recommendation**:

Change the format of array to
“ s = new Stake[](_to - _from+1)” or
“for (uint256 i = _from; i < _to; i++)”
