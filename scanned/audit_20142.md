# Unsafe calculation

## Summary
This bug report is about a function called massHarvest in a contract called YieldFarm.sol. The function has a calculation that does not have a safe math check, which could lead to potential issues. The recommendation is to add a safe math check to ensure the calculation is done correctly.

## Details
**Description**

In contract YieldFarm.sol (argo-yieldfarming-staking-master), in function massHarvest there is
a calculation without safe math check:
totalDistributedValue += _harvest(i);

**Recommendation**:

Add safe math:
totalDistributedValue = totalDistributedValue.add(_harvest(i));
