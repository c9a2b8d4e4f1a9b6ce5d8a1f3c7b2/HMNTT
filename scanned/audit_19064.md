# Possible remove of necessary adapter

## Summary
A bug has been discovered in the Gearbox Protocol contracts, which is related to the use of an adapter with multiple contracts. If an adapter is used for several contracts, then the next lines in the CreditFilter.sol file will break the work of these contracts. To fix this issue, it is recommended to add a check that removed adapter is not used in other contracts.
