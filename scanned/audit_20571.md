# No tests.

## Summary
This bug report states that there is a lack of test coverage for the formal logic of contracts, which means that the functionality of the contracts cannot be guaranteed to be correct. The recommendation is to provide initial test coverage for the formal logic in order to ensure the correctness of the contracts.

## Details
**Description**

Absent test coverage. Formal logic is not checked, so currently it cannot guarantee the
correctness of contracts functionality.

**Recommendation**:

Provide the initial test coverage for the formal logic of the contracts
