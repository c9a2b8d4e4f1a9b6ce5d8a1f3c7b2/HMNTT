# Balance not checked

## Summary
This bug report is about the balance of the creditAccount not being checked in the Gearbox Protocol. The Gearbox Protocol is a decentralized protocol for creating and managing digital assets. 

The code in question is located at https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/adapters/YearnV2.sol#L83. The recommendation is to add a check that balanceBefore is greater than 0.
