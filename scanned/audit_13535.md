# initialize – Committee members array can contain duplicates ✓ Fixed

## Summary
This bug report is about an issue with the pool's `initialize` function in the code/contracts/Pool.sol file. The array of initial committee members is given as an argument to the function, but it does not check for duplicates. This can lead to the discrepancy between the length of the array and the actual number of committee members, which could cause more problems such as an insufficient committee size to reach the threshold. The bug was fixed by implementing a nested loop to de-duplicate committee members. The `initialize` function should also verify that `member` hasn't been added before, using the same technique as `_executeAddToCommittee` which refuses to add someone who is already in the committee.

## Details
#### Resolution



Fixed in [3bbafab926df0ea39f444ef0fd5d2a6197f99a5d](https://github.com/TidalFinance/tidal-contracts-v2/tree/3bbafab926df0ea39f444ef0fd5d2a6197f99a5d) by implementing a nested loop to de-duplicate committee members.


#### Description


The initial committee members are given as array argument to the pool’s `initialize` function. When the array is processed, there is no check for duplicates, and duplicates may also end up in the storage array `committeeArray`.


**code/contracts/Pool.sol:L43-L47**



```
for (uint256 i = 0; i < committeeMembers\_.length; ++i) {
 address member = committeeMembers\_[i];
 committeeArray.push(member);
 committeeIndexPlusOne[member] = committeeArray.length;
}

```
Duplicates will result in a discrepancy between the length of the array – which is later interpreted as the number of committee members – and the actual number of (different) committee members. This could lead to more problems, such as an insufficient committee size to reach the threshold.


#### Recommendation


The `initialize` function should verify in the loop that `member` hasn’t been added before. Note that `_executeAddToCommittee` refuses to add someone who is already in the committee, and the same technique can be employed here.
