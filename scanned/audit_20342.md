# Unused field in Package

## Summary
The bug report suggests that the "status" field in the package structure is not being used and can be removed to save storage space. The recommendation is to replace the package structure with a mapping that tracks maximum payout numbers.

## Details
**Description**
Field “status” of package structure is unused. In order to decrease storage usage is better to
remove it and replace mapping for Packages to mapping for maxPayout numbers.
**Recommendation**:
Remove Package structure and use mapping(uint=>uint) to track max payouts.
