# Smart contract owner is able to change phase of sale anytime.

## Summary
The CryptoBearWatchClub.sol smart contract has a bug in the setSaleStatus function. This allows the owner of the contract to change the sale phase without any restrictions. A recommendation is to restrict the owner's ability to change sale phases.

## Details
**Description**

CryptoBearWatchClub.sol, setSaleStatus

**Recommendation**:

Restrict Smart contract owner to go over different sale phases.
