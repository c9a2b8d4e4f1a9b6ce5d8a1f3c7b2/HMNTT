# Incorrect usage of function returned value

## Summary
The report states that there is a bug in the withdraw function of Vault. Currently, the function returns the amount of tokens transferred instead of the shares. The recommendation is to change the code of the function.

## Details
##### Description
Vault's withdraw function returns amount of tokens which was transferred, not shares:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/adapters/YearnV2.sol#L145
##### Recommendation
We recommend to change function code.
