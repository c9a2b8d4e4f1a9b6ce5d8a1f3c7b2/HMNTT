# Owner is able to withdraw reward tokens.

## Summary
The bug report is about a function called "recoverERC20()" in a contract called "Bribes.sol". The report states that the owner of the contract is able to withdraw any tokens, including reward tokens, which can cause problems with distributing rewards. This could be used by malicious people if the owner is a DAO contract. The recommendation is to make sure that reward tokens cannot be withdrawn.

## Details
**Description**

Bribes.sol: function recoverERC20().
Owner is able to withdraw any tokens from the contract, including reward tokens, which
might lead to errors in reward distribution. In case, owner is set to a DAO contract, this can
potentially be used by malicious actors.

**Recommendation**:

Validate that reward tokens cannot be withdrawn.
