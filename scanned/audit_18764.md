# View function will never return value.

## Summary
The bug report is about a problem in a code called StakingB.sol, specifically in line 46. The issue is that a function called "stakesInfo" is not returning any value, which is caused by an error in the code. The code creates an array that is supposed to be iterated through, but the way it is set up is incorrect. The report suggests two possible solutions to fix the issue: changing the format of the array or using a different loop method.

## Details
**Description**

StakingB.sol Line 46
“stakesInfo” will never return value because it creates an array [from; to) but tries to iterate
through [from; to].

**Recommendation**:

Change the format of array to
“ s = new Stake[](_to - _from+1)” or
“for (uint256 i = _from; i < _to; i++)”
