# Smart contract owner is able to mint arbitrary number of NFT tokens.

## Summary
The CryptoBearWatchClub.sol smart contract has a bug where the owner is able to create an unlimited amount of tokens. This could potentially lead to inflation and devalue the tokens. It is recommended to limit the owner's ability to mint tokens in order to prevent this issue.

## Details
**Description**

CryptoBearWatchClub.sol, reserveBears

**Recommendation**:

Limit ability of Smart contract owner to mint tokens.
