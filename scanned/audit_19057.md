# Unnecessary allowance

## Summary
This bug report focuses on the Vault's withdraw function in the Gearbox Protocol. The withdraw function is found in the YearnV2.sol file and is burning shares, meaning that allowance is not necessary. The recommendation is to remove providing allowance. In other words, the Vault's withdraw function should be modified so that it does not require allowance, and instead burns the shares without it. This bug report is important for anyone using the Gearbox Protocol, as it ensures that the Vault's withdraw function is working correctly and efficiently.

## Details
##### Description
Vault's withdraw function burns shares, so allowance is unnecessary:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/adapters/YearnV2.sol#L130
##### Recommendation
We recommend removing providing allowance.
