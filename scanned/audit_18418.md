# DRY. Rewards calculation logic is placed in both methods claimReward and getTotalClaimable.

## Summary
This bug report is about a specific code in a program called Arkouda.sol. The code has three methods: claimReward, getTotalClaimable, and a recommendation to improve the code. The recommendation suggests that the method getTotalClaimable should be used inside the claimReward method to make the code more efficient. This will also reduce the amount of code needed.

## Details
Arkouda.sol, claimReward and getTotalClaimable

**Recommendation**:
Consider to invoke method getTotalClaimable inside of claimReward to reuse existent logic
and decrease final byte code.
