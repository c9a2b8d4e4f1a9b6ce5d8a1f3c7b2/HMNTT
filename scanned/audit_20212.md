# Solidity version update

## Summary
This bug report states that there is an issue with the currently used version (0.6.1) of a smart contract. This issue is classified as medium and is considered a vulnerability. The report recommends updating the solidity version to the latest one, such as 0.6.12, 0.7.6, or 0.8.4, to address this issue.

## Details
**Description**

Issue is classified as Medium, because it is included to the list of standard smart contracts’
vulnerabilities. Currently used version (0.6.1) is not the last in the line, so it contradicts
standard checklist.

**Recommendation**:

You need to update the solidity version to the latest one in the branch - consider 0.6.12, 0.7.6
or 0.8.4.
