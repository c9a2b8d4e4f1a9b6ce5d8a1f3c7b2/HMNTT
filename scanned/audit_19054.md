# Incorrect minting

## Summary
This bug report concerns the transferFrom function in the USDT contract. The issue is that the contract is not taking into account the fees associated with the transfer, resulting in the contract having less than the intended amount.

The recommendation is to use the balance difference to mint Diesel tokens. This will ensure that the fees are accounted for and the contract will have the intended amount.

In conclusion, this bug report concerns the transferFrom function in the USDT contract and the recommendation is to use the balance difference to mint Diesel tokens.
