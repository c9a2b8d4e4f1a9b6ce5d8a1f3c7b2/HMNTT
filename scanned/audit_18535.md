# Smart contract owner is able to mint arbitrary number of NFT tokens.

## Summary
The report describes an issue with a specific smart contract called CryptoBearWatchClub.sol. The problem involves the function reserveBears, which allows the contract owner to create new tokens. The recommendation is to limit the owner's ability to mint tokens in order to prevent potential misuse of the contract.

## Details
**Description**

CryptoBearWatchClub.sol, reserveBears

**Recommendation**:

Limit ability of Smart contract owner to mint tokens.
