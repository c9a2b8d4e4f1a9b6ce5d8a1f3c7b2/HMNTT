# [H-01] setGuardian() Wrong implementation

## Summary
This bug report is about two contracts, IbbtcVaultZap.sol and SettToRenIbbtcZap.sol, which are part of the Badger-Finance/badger-ibbtc-utility-zaps project. In both contracts, the line `governance = _guardian` should be `guardian = _guardian`. This line of code is used to set the guardian address, which is used to ensure that the contract functions as intended. 

The bug was discovered by the user WatchPug and the code is available for review at the links provided in the report. 

It is important to fix this bug as soon as possible, as it could lead to unintended behavior in the contracts. If the incorrect line of code is left unchanged, the contract could be vulnerable to malicious actors, which could cause serious financial damage.
