# Absent framework.

## Summary
This bug report states that there is a problem with the development and deployment processes due to the lack of a framework (specifically Truffle or Brownie) and additional migrations or scripts. The recommendation is to provide an initial framework for the contracts set in order to address this issue.

No framework (Truffle or Brownie) and additional migrations/scripts - it affects further
development and deployment processes.

**Recommendation**:

Provide the initial framework for the contracts set
