# Presale round does not return ETH

## Summary
There is an issue with the CryptoBearWatchClub contract's presaleMint() function, where it does not return excess ETH like the mint() function does. This means that ETH may become stuck in the contract. It is recommended to verify if this is intended behavior or to correct it.

## Details
**Description**

CryptoBearWatchClub.sol, presaleMint()
presaleMint() function does not return excess ETH amount like the mint() function does. Thus
ETH may be left stuck on the contract.

**Recommendation**:

Verify the presaleMint() should not return excess ETH or correct the behaviour.
