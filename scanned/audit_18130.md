# Function can’t be called.

## Summary
The bug report is about a function called runDisableStrategy() that can only be called by a controller. However, there is no implementation for calling this function in Controller.sol. The recommendation is to add an implementation for calling the function spool.runDisableStrategy() from the controller.

## Details
**Description**

Function runDisableStrategy(). Function can be called by controller only, however there is not
implementation for calling this function in Controller.sol

**Recommendation**:

Add an implementation of call function spool.runDisableStrategy() from controller.
