# [H-01] Lack of withdrawal limits check

## Summary
The bug report discusses a recent update that has caused a medium severity issue with a high likelihood of occurrence. The update has removed withdrawal limit checks from the `CollateralConfiguration.checkWithdrawLimits` function and placed them in the `AccountCollateral.updateBalance` function. This has resulted in the `AccountCollateral.transferFunds` and `ExposedModule.updateCollateralShares` functions bypassing these checks, which could potentially cause problems. The report recommends implementing checks for these functions to address the issue.

## Details
**Severity**

**Impact:** Medium

**Likelihood:** High

**Description**

The provided update fully excludes withdrawal limits checks from the `CollateralConfiguration.checkWithdrawLimits` function and puts them into the `AccountCollateral.updateBalance` function. So `AccountCollateral.transferFunds` and `ExposedModule.updateCollateralShares` functions have neither `GlobalCollateralConfiguration.checkWithdrawLimits` nor `CollateralConfiguration.checkWithdrawLimits` check since they bypass the `updateBalance` function.
