# Smart contract owner is able to change phase of sale anytime.

## Summary
The bug report is about a smart contract called CryptoBearWatchClub.sol, specifically the function setSaleStatus. The recommendation is to restrict the owner of the contract from being able to change the sale phases.

## Details
**Description**

CryptoBearWatchClub.sol, setSaleStatus

**Recommendation**:

Restrict Smart contract owner to go over different sale phases.
