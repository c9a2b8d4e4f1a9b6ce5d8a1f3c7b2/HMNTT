# Burn functions should be removed

## Summary
This bug report suggests that the burn function in the white paper should be removed as it is not necessary for the core ERC20 token standard. The report recommends following the guidelines stated in the white paper, which states that there will be no burning of tokens.

## Details
**Description**

Since the white paper states that, there would never be any burning of tokens, we advise the
burn functions be removed as it’s not a required function of the core ERC20 token standard.
