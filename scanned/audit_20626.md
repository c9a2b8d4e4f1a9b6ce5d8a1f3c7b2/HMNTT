# [H-03] Missing DAO functionality to call changeDAO() function in Vader.sol

## Summary
This bug report describes a vulnerability in the Vader.sol smart contract. The changeDAO() function is authorized to be called only from the DAO, but the DAO contract has no corresponding functionality to call changeDAO(). As a result, the DAO address cannot be changed. This was discovered through manual analysis and a proof of concept can be found at the given link. The recommended mitigation step is to add functionality to the DAO to call the changeDAO() function of Vader.sol.
