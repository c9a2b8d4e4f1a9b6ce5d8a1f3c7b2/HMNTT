# [H-05] Withdrawable.withdraw does not decrease pendingWithdrawals

## Summary
This bug report is about an issue with the FSD.mint calculation in the code. The name 'pendingWithdrawals' indicates that the storage variable tracks withdrawals that need to be paid out, which also matches the behavior in '_increaseWithdrawal'. However, it is not decreased when withdrawing in 'withdraw'. This causes the 'getReserveBalance' to consistently under-report the actual reserve balance, leading to wrong mint amounts being used in the FSD.mint calculation. The recommendation is to decrease 'pendingWithdrawals' by the withdrawn amount.
