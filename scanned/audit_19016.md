# Incorrect update of list

## Summary
This bug report concerns an issue with a list not being updated properly in the Gearbox Protocol's AccountFactory.sol file. The link to the specific line of code is provided in the report. It is recommended that the line of code `_nextCreditAccount[creditAccount] = address(0);` be added in order to resolve this issue.

## Details
##### Description
List isn't updated properly:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/core/AccountFactory.sol#L260
##### Recommendation
We recommend to add `_nextCreditAccount[creditAccount] = address(0);`
