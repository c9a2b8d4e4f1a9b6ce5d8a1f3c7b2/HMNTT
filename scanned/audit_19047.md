# `priceFeeds` can't be changed

## Summary
This bug report is about the PriceOracle smart contract which is part of the Gearbox Protocol. The issue is that the `priceFeeds` variable cannot be changed. This variable is used to set the price for various assets in the protocol. The recommendation is to add a function to the smart contract that allows the `priceFeeds` variable to be changed. This would enable the protocol to update the prices of assets as needed.

## Details
##### Description
`priceFeeds` can't be changed:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/oracles/PriceOracle.sol#L56
##### Recommendation
We recommend adding a function for changing `priceFeeds`.
