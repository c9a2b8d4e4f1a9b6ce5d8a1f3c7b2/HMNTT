# Incorrect input parameters

## Summary
This bug report is about the Gearbox Protocol, a decentralized finance protocol. The bug is that input data is not checked for possible containing of zero addresses. This could lead to errors in the protocol's code.

The report lists twenty-one lines of code in the Gearbox Protocol's GitHub repository where the bug can be found. The code is spread across twelve different files.

The recommendation is to add a check that input data is not equal to zero address. This will prevent errors from occurring in the protocol's code.

Overall, this bug report is about a bug in the Gearbox Protocol's code. The bug is that input data is not checked for possible containing of zero addresses. The report lists the lines of code where the bug can be found and recommends adding a check that input data is not equal to zero address.
