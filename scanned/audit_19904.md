# Use ReentrancyGuard for every external method

## Summary
This bug report suggests that in order to prevent reentry attacks, a nonReentrant modifier should be used for every method that changes the state. The recommendation is to use this modifier for all external methods to ensure safety.

## Details
**Description**

To completely prevent reentry attacks it's better to place a nonReentrant modifier for every
state changing method.

**Recommendation**:

Go safe. Use nonReentrant for every external method.
