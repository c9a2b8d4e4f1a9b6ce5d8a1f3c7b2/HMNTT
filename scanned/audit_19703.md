# [M-04] Fee on transfer tokens do not work within the protocol

## Summary
This bug report is about an issue with the handling of transfer tokens in the 2021-10-defiprotocol. Specifically, the protocol is transferring less tokens than expected. This is due to the protocol requesting incorrect amounts when dealing with these tokens. The resolution to this issue is for the protocol to use stored token balances instead of transfer for calculating amounts. This bug report contains a link to the specific code that needs to be changed in order to fix the issue.

## Details
