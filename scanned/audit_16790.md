# `onlyAdmin` guard on private function `_addRewards` in MarinateReceiver.sol.

## Summary
The bug report is about a problem with the code in a file called MarinateReceiver.sol. There is a function called _addRewards that is supposed to be accessible by both an admin and automated bots, but the code only allows the admin to access it. This means that the bots cannot use this function, causing issues with another function called sendBalancesAsRewards. The recommendation is to remove the code that restricts access to only the admin, as it is not necessary since the function is already private. The bug has been fixed by removing the code.
