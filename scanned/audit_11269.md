# Lack of Event Emission

## Summary
The bug report states that certain functions in various contracts are not emitting relevant events after executing sensitive actions. This makes it difficult to track and monitor the contracts' activity. The report suggests emitting events after these changes are made to improve tracking and notify off-chain clients. The issue has since been resolved in a recent pull request.
