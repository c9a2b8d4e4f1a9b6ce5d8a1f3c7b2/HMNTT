# Solidity version update

## Summary
The bug report states that there is an issue with the current version (0.8.0) of a smart contract. This issue is classified as medium and is listed as a vulnerability in the standard checklist. The recommendation is to update the solidity version to the latest one (0.8.4) in order to address this issue.

## Details
**Description**

Issue is classified as Medium, because it is included to the list of standard smart contracts’
vulnerabilities. Currently used version (0.8.0) is not the last in the line, so it contradicts
standard checklist.

**Recommendation**:

You need to update the solidity version to the latest one in the branch - consider 0.8.4.
