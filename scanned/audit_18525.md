# [H-01] Update initializer modifier to prevent reentrancy during initialization

## Summary
This bug report is about a vulnerability in the Governable.sol contract, which is used in multiple places in the code. The vulnerability is due to the use of an outdated version of the "@openzeppelin/contracts" dependency. This vulnerability could allow attackers to exploit the `initializer()` function in the contracts AMM.sol, ClearingHouse.sol, MarginAccount.sol, and Oracle.sol.

In order to mitigate this vulnerability, it is recommended to upgrade the "@openzeppelin/contracts" dependency to version 4.4.1 or higher. This should be done as soon as possible in order to prevent any potential exploits.
