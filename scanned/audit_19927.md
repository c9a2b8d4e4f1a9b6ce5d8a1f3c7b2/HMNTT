# The contract owner can set any fee

## Summary
The updateMaxFee() function in this contract allows the owner to set a fee amount that can be higher than 100%, which is not intended. To fix this, a constant variable can be added to set a maximum fee and ensure that the function checks for this condition before updating the fee amount.

## Details
**Description**

Function updateMaxFee() allows the owner of the contract to set any value for the fee amount
even greater than the 100%.

**Recommendation**:

You can add a constant variable for max fee and verify that this condition is met in function
updateMaxFee().
