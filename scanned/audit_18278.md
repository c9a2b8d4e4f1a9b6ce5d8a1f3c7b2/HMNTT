# Owner is able to withdraw any token balance without restrictions.

## Summary
The bug report discusses a potential issue with a function called "emergencyWithdraw" in a smart contract. This function allows the owner of the contract to withdraw funds from users, which is considered a security risk. The recommendation is to validate that the owner cannot withdraw all funds at any time. After further discussion with the client, it was determined that this is not a bug, but a deliberate design. The function can only be called by a specific user and will eventually be delegated to a timelock/governance system for added security.
