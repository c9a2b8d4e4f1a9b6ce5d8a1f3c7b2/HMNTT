# Incorrect parameter passed

## Summary
This bug report is about the Curve swap function in the Gearbox Protocol. The function is currently returning more than the minimum yield (min_dy). The recommendation is to use the balance difference as a parameter for the function. This will ensure that the function returns the correct amount of yield.

## Details
##### Description
Curve swap can return more than `min_dy`:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/adapters/CurveV1.sol#L82
##### Recommendation
We recommend to use balance difference as a parameter for the function.
