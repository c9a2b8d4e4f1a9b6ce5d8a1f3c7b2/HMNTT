# Unnecessary allowance

## Summary
The report discusses a bug in the withdraw function of Vault, which results in the burning of shares. This bug is caused by an unnecessary allowance in the code. The recommendation is to remove this allowance to fix the bug.

## Details
##### Description
Vault's withdraw function burns shares, so allowance is unnecessary:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/adapters/YearnV2.sol#L130
##### Recommendation
We recommend removing providing allowance.
