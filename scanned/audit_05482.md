# Tests do not work

## Summary
This bug report states that the tests for the protocol are not functioning, which means that it is not possible to check the test coverage and create proof of concepts. The recommendation is to make fixing the tests a priority and ensure they are set up properly before deploying the protocol.

## Details
##### Description
Currently, the protocol's tests do not work, making it impossible to check test coverage and prepare PoCs.

##### Recommendation
We recommend prioritizing, preparing, and setting up all the tests before the protocol deployment.
