# Incorrect usage of function returned value

## Summary
This bug report is about the Vault's withdraw function in the Gearbox Protocol. The function is currently returning the amount of tokens which was transferred, rather than the amount of shares. This is an issue because it is not accurate. It is recommended to change the function code in order to fix the bug.

## Details
