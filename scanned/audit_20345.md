# Unsafe operations

## Summary
This bug report highlights a potential issue with the use of "unsafe" "+" and "+=" operations in a program. The report suggests using SafeMath or updating to Solidity 0.8.6 to ensure safer operations. The recommendation is to use SafeMath for all instances of these operations.

## Details
**Description**
There are a lot of unsafe “+” and “+=” operations. Consider SafeMath usage or Solidity 0.8.6

**Recommendation**:
Use SafeMath for all places where “+” and “+=” operation is used.
