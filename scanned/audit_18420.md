# Methods accepting empty array as input arguments.

## Summary
The bug report is about a problem with four functions in a file called Arkouda.sol. These functions are stakeCBWC, unStakeCBWC, and claimRewards. The report recommends adding a check to make sure the array of token ids has a length greater than zero. This will help prevent the issue from occurring.

## Details
**Description**


Arkouda.sol, stakeCBWC, unStakeCBWC, claimRewards

**Recommendation**:

Add check that the array of token ids has length more than zero.
