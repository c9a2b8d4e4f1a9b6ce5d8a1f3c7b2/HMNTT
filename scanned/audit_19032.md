# User can approve any token

## Summary
This bug report is about the CreditManager.sol contract, which is part of the Gearbox Protocol. It has been discovered that the contract allows any user to approve any token. This means that the user can approve tokens that are not allowed, which can cause problems. 

To fix this issue, it is recommended that a check is added to the contract so that only allowable tokens can be approved. This will help ensure that only the correct tokens are approved, and that any tokens that are not allowed cannot be approved. 

In conclusion, this bug report highlights an issue with the CreditManager.sol contract, which allows any user to approve any token. To fix this, it is recommended that a check is added to the contract so that only allowable tokens can be approved.
