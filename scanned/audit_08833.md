# [M-05] Guardian can't unpause the RelayerV2

## Summary
The report is about a bug in the Guardian protocol, which is used to pause and unpause certain functions. According to the documentation, Guardian should be able to pause and unpause the protocol, but the bug prevents it from doing so. This affects the functionality of the protocol and is considered a medium severity issue with a medium likelihood of occurring. The recommended solution is to allow Guardian to unpause the protocol.

## Details
**Severity**

**Impact:** Medium

**Likelihood:** Medium

**Description**

According to the docs, Guardian should be able to pause/unpause the protocol.

> Guard : Able to pause/unpause the protocol and quarantine assets.

The issue is that the unpause functions access controls don't allow Guardian to unpause the protocol.

```solidity
  ///@notice Unpause the protocol
  function unpauseProtocol() external onlyOwner {
    paused = false;
    emit Unpause(block.timestamp);
  }

  ///@notice Unpause the swaps
  function unpauseSwap() external onlyOwner {
    swapPaused = false;
    emit Unpause(block.timestamp);
  }
```

**Recommendations**

Allow Guardian to unpause the protocol.
