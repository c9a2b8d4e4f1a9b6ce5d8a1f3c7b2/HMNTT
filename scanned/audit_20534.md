# Using + instead of add.

## Summary
The bug report is advising users to use the "add" method from the safeMath library to avoid potential overflow issues. The report is suggesting this as a precautionary measure to prevent any potential problems that may occur.

## Details
![image](https://github.com/user-attachments/assets/8614db7d-1497-4d9c-b9cf-fbfe56f78704)

**Recommendation**:
Use the add method of the safeMath library, as in theory an overflow is possible.
