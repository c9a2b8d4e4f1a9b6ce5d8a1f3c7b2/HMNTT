# SafeERC20 usage preferable

## Summary
This bug report is about a function called TokenRecover which is used to recover tokens. The report suggests using a library called SafeTransfer to avoid an error that can occur when transferring certain types of tokens. The recommendation is to use a different library called SafeERC20 to prevent this error.

## Details
TokenRecover, recoverERC20()

**Description**

Consider usage of SafeTransfer library, because unsafe token transfer will revert with the non-
standard implementation of the ERC20 standard (like for example USDT has).

**Recommendation**:

Use SafeERC20 library.
