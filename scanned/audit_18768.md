# View function will never return value.

## Summary
This bug report discusses an issue with the code in line 46 of the StakingB.sol file. It states that the "stakesInfo" function will not return a value because it creates an array that is not properly iterated through. The recommendation is to change the format of the array or use a for loop to properly iterate through the array.

## Details
**Description**

StakingB.sol Line 46
“stakesInfo” will never return value because it creates an array [from; to) but tries to iterate
through [from; to].

**Recommendation**:

Change the format of array to
“ s = new Stake[](_to - _from+1)” or
“for (uint256 i = _from; i < _to; i++)”
