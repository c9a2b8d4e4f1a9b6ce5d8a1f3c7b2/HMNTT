# Admin can drain all rewards from all incentives

## Summary
This bug report is about an issue with the `AlgebraEternalFarming.decreaseRewardsAmount()` function in the Algebra repository. This function sends rewards to the address of the admin, which could lead to a rugpull, meaning the admin could take all of the rewards away from users. This would reduce the trust in the incentive system. The bug report recommends changing the logic of rewards refunds so that an incentive for specific farming is controlled by a registered address for that incentive.

## Details
##### Description

`AlgebraEternalFarming.decreaseRewardsAmount()` sends rewards to the address of admin:
https://github.com/cryptoalgebra/Algebra/blob/7290fad656bfa89db3743c52af631154f6a8a2d5/src/tokenomics/contracts/farmings/AlgebraEternalFarming.sol#L172.
It creates the danger of a rugpull and reduces the trust in the incentive system.

##### Recommendation
We recommend changing the logic of rewards refunds. An incentive for specific farming should be controlled by a registered address for this incentive.
