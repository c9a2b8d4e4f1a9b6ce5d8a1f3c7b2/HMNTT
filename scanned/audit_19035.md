# Usage of ERC777 token can block liquidation

## Summary
This bug report is about how malicious users can use ERC777 token callbacks to block liquidation. This type of token is used in the Gearbox Protocol, which is a decentralized protocol for tokenized debt. The bug report states that malicious users can use the ERC777 token callbacks to block liquidation. This means that the malicious user can prevent the liquidation of debt, which is an important part of the protocol. The report recommends not to use ERC777 token as the underlying token, as this would prevent the malicious user from blocking liquidation. In conclusion, this bug report highlights a potential security issue with the Gearbox Protocol and provides a recommendation for how to address it.
