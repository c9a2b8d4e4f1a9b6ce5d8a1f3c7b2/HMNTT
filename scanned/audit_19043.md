# Too many rights for configurator

## Summary
This bug report is about the Configurator, a part of the Gearbox Protocol, which is a set of contracts that can be used to manage digital assets. The report states that the Configurator has too many rights and can potentially steal all user's funds. To prevent this, the report recommends using a DAO (Decentralized Autonomous Organization) for the Configurator. This would ensure that the Configurator does not have too many rights and that user's funds are protected.

##### Description
