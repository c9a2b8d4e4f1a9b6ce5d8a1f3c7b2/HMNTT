# View function will never return value.

## Summary
The bug report is about a coding error in a file called StakingB.sol, specifically on line 46. The issue is that a function called "stakesInfo" is not returning any values because it is trying to iterate through an array in a way that will not work. The recommendation is to change the format of the array or use a different method to iterate through it.

## Details
**Description**


StakingB.sol Line 46
“stakesInfo” will never return value because it creates an array [from; to) but tries to iterate
through [from; to].

**Recommendation**:

Change the format of array to
“ s = new Stake[](_to - _from+1)” or
“for (uint256 i = _from; i < _to; i++)”
