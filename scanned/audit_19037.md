# Transfer to zero address

## Summary
This bug report is about the code in the CreditManager.sol file, located at the given URL. The code allows the variable "to" to be equal to the zero address. The recommendation is to add a check that "to" is not equal to the zero address. This will help to prevent any potential security issues that could arise from this code.

## Details
##### Description
`to` can be equal to zero address:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/credit/CreditManager.sol#L338
##### Recommendation
We recommend adding a check that `to != address(0)`.
