# Centralization risk and possibility of a Denial of Service

## Summary
Bug Report Summary: A bug has been discovered where a malicious admin can drain all the deposited tokens from the contract by calling cancelLauncher(). The bug has been resolved, and it is recommended to add a multisig or governance mechanism to prevent this from happening in the future.

## Details
**Severity**: Medium

**Status**: Resolved

**Description**

A malicious admin can call cancelLauncher() and drain all the deposited tokens from the contract. 

**Recommendation**: 

It is advised to add a multisig or a governance mechanism to prevent this from happening.
