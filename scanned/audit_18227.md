# View function will never return value.

## Summary
The bug report is about a problem with a line of code in a file called StakingB.sol. The code creates an array that is used to store information, but the way it is set up means that the information cannot be accessed properly. The report recommends changing the format of the array or using a different method to iterate through it.

## Details
**Description**

StakingB.sol Line 46
“stakesInfo” will never return value because it creates an array [from; to) but tries to iterate
through [from; to].

**Recommendation**:

Change the format of array to
“ s = new Stake[](_to - _from+1)” or
“for (uint256 i = _from; i < _to; i++)”
