# [M-03] Setting Factory.bondPercentDiv to zero cause Denial of Service in Auction.bondForRebalance()

## Summary
This bug report is about a vulnerability in the function `Factory.setBondPercentDiv()` which allows the owner to set the state variable `Factory.bondPercentDiv` to zero. If `Factory.bondPercentDiv` equals zero then the function `Auction.bondForRebalance()` will always revert due to a division by zero. The tool used to identify this vulnerability was a manual code review. The recommended mitigation steps for this vulnerability are to add an appropriate require statement to the function `Factory.setBondPercentDiv()` to disallow setting `Factory.bondPercentDiv` to zero.

## Vulnerability details

The function `Factory.setBondPercentDiv()` allows the owner to set the state variable `Factory.bondPercentDiv` to zero.

## Impact
If `Factory.bondPercentDiv` equals zero then the function `Auction.bondForRebalance()` will always revert due to a division by zero:
```
bondAmount = basketToken.totalSupply() / factory.bondPercentDiv();
```

## Tool Used
Manual code review.

## Recommended Mitigation Steps
Add an appropriate require statement to the function `Factory.setBondPercentDiv()` to disallow setting `Factory.bondPercentDiv` to zero.
