# Account can be opened for zero address

## Summary
This bug report is about the Gearbox Protocol Credit Manager Contract. Currently, the `onBehalfOf` parameter can be equal to a zero address, which is not allowed. The recommendation is to add a check to make sure that `onBehalfOf` is not equal to a zero address. This check should be added to the Credit Manager Contract to ensure that the parameter is not equal to a zero address. This will help to prevent any issues that may arise from having an invalid parameter.

## Details
##### Description
`onBehalfOf` can be equal to zero address:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/credit/CreditManager.sol#L180
##### Recommendation
We recommend adding a check that `onBehalfOf` is not equal to zero address.
