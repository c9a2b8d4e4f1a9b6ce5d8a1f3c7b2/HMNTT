# `masterCreditAccount` remains uninitialized

## Summary
This bug report is about the `masterCreditAccount` in the Gearbox Protocol contracts. This account has not been initialized and is vulnerable to anyone being able to initialize it. It is recommended to initialize the `masterCreditAccount` in order to secure it and prevent any malicious activity.

## Details
##### Description
`masterCreditAccount` remains uninitialized, so anybody can initialize it:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/core/AccountFactory.sol#L90
##### Recommendation
We recommend to initialize `masterCreditAccount`.
