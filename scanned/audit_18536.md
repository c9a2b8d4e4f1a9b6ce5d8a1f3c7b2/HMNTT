# Presale round does not return ETH

## Summary
The CryptoBearWatchClub.sol contract has a bug in the presaleMint() function. Unlike the mint() function, presaleMint() does not return excess ETH, which means that ETH can get stuck in the contract. It is recommended to check and fix this issue.

## Details
**Description**

CryptoBearWatchClub.sol, presaleMint()
presaleMint() function does not return excess ETH amount like the mint() function does. Thus
ETH may be left stuck on the contract.

**Recommendation**:

Verify the presaleMint() should not return excess ETH or correct the behavior.
