# Incorrect event emitting

## Summary
This bug report is about an issue with the code in the smart-contracts repository of FantiumAG. Specifically, the event on line 386 of the FantiumNFTV3.sol file should be changed from its current form to `emit Mint(_to, tokenId + i);`. This change would ensure that the event is correctly emitted and that the code runs as intended.

## Details
##### Description
Here https://github.com/FantiumAG/smart-contracts/blob/a2d126453c1105028f12277b8f342d2cdbf01a77/contracts/FantiumNFTV3.sol#L386 the event should be `emit Mint(_to, tokenId + i);`.

##### Recommendation
We recommend changing the event to the `emit Mint(_to, tokenId + i);`.
