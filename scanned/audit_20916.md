# Gas overflow during iteration (DoS)

## Summary
This bug report is about a gas flow issue that occurs in the Grandthrax/yearnV2-generic-lender-strat. It is caused when more gas is required than allocated to record one block. This affects the line 413 in the Strategy.sol file. To resolve this issue, it is recommended to add a check for the maximum possible number of elements of the arrays.

## Details
##### Description
Each iteration of the cycle requires a gas flow.
A moment may come when more gas is required than it is allocated to record one block. In this case, all iterations of the loop will fail.
Affected lines:
https://github.com/Grandthrax/yearnV2-generic-lender-strat/blob/979ef2f0e5da39ca59a5907c37ba2064fcd6be82/contracts/Strategy.sol#L413

##### Recommendation
It is recommended adding a check for the maximum possible number of elements of the arrays.
