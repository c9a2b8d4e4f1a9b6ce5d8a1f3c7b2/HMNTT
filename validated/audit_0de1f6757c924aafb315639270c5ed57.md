Audit Report

## Title
Historical Block Parameter Accepted for Gas Estimation, Binary Search Executes Against Stale State

## Summary
`ContractController.constructServiceParameters()` accepts `estimate: true` alongside a user-supplied historical `block` value with no cross-validation. Both fields are forwarded verbatim into `ContractExecutionParameters`. `ContractExecutionService.estimateGas()` then performs its initial probe call and the entire binary search against the historical Hashgraph state corresponding to that block, returning a gas figure that is invalid for the current ledger state.

## Finding Description

**Entry point — no guard on `estimate=true` + historical block:**

`ContractController.constructServiceParameters()` reads both fields and passes them directly to the builder with no cross-validation: [1](#0-0) 

`ContractCallRequest` has two `@AssertTrue` validators — `hasFrom()` and `hasTo()` — that only check value/sender/receiver relationships. Neither rejects the combination of `estimate=true` with a historical block: [2](#0-1) 

**Historical context is set once and never cleared during the binary search:**

`ContractCallService.callContract()` sets the block supplier on the shared `ContractCallContext` from `params.getBlock()`: [3](#0-2) 

`ContractCallContext.useHistorical()` returns `true` whenever `block != BlockType.LATEST`, which drives all state reads to the historical snapshot: [4](#0-3) 

**`BinaryGasEstimator.search()` resets only the write-cache, not the block context:**

Each iteration calls `contractCallContext.reset()`: [5](#0-4) 

`ContractCallContext.reset()` only clears `writeCache`; `blockSupplier` and `callServiceParameters` (which carries the historical `block`) are untouched: [6](#0-5) 

**`ContractExecutionService.estimateGas()` passes the same `params` (with historical block) into the binary search lambda:** [7](#0-6) 

The lambda `gas -> doProcessCall(params, gas, true)` captures `params` by closure, so every EVM execution inside the binary search reads from the historical snapshot.

**`BlockType.of()` accepts any non-negative integer or hex block number without restriction:** [8](#0-7) 

## Impact Explanation
The returned hex value is a gas figure derived entirely from a past ledger state. If the historical state had lower storage-write costs (e.g., a slot was already warm/non-zero at that block but is cold/zero today), the estimate will be below the gas actually required for the current transaction, causing on-chain `INSUFFICIENT_GAS` failures and loss of transaction fees for users who rely on the estimate. Conversely, a block where a contract did not yet exist may cause the estimate call to succeed vacuously and return an artificially low value. The mirror node's contract is to return estimates valid for the current state; this breaks that contract silently with no error to the caller.

## Likelihood Explanation
The attack requires zero privileges: a single unauthenticated POST to `/api/v1/contracts/call` with `"estimate": true` and any valid historical block number (e.g., `"block": "0x1"`). `BlockType.of()` accepts any non-negative integer or hex value. The attacker needs only to know a block number where the target contract's gas profile differed from today, which is public information via the mirror node's own block/transaction APIs. The issue is trivially repeatable and requires no special tooling.

## Recommendation
Add a cross-validation constraint — either in `ContractCallRequest` (as an `@AssertTrue` method) or in `ContractController.constructServiceParameters()` — that rejects any request where `estimate=true` and `block != BlockType.LATEST`. A clear error response (e.g., HTTP 400 with a message such as `"Gas estimation is only supported for the latest block"`) should be returned. This prevents the binary search from ever being seeded with a historical block context.

## Proof of Concept
```
POST /api/v1/contracts/call
Content-Type: application/json

{
  "to": "<deployed_contract_address>",
  "data": "<call_data>",
  "estimate": true,
  "block": "0x1"
}
```
The response will be a hex-encoded gas figure computed entirely against the state at block 1. No authentication is required. The caller receives no warning that the estimate is stale.

    private Bytes estimateGas(final ContractExecutionParameters params, final ContractCallContext context) {
        final var processingResult = callContract(params, context);
        final var gasUsedByInitialCall = processingResult.gasUsed();

        // sanity check ensuring gasUsed is always lower than the inputted one
        if (gasUsedByInitialCall >= params.getGas()) {
            return Bytes.ofUnsignedLong(gasUsedByInitialCall);
        }

        final var status = ResponseCodeEnum.SUCCESS.toString();
        final var estimatedGas = binaryGasEstimator.search(
                (totalGas, iterations) -> updateMetrics(params, totalGas, iterations, status),
                gas -> doProcessCall(params, gas, true),
                gasUsedByInitialCall,
                params.getGas());

        return Bytes.ofUnsignedLong(estimatedGas);

    public static BlockType of(final String value) {
        if (StringUtils.isEmpty(value)) {
            return LATEST;
        }

        final var blockTypeValue = value.toLowerCase(Locale.ROOT);
        final var matcher = BLOCK_PATTERN.matcher(blockTypeValue);
        if (!matcher.matches()) {
            throw new IllegalArgumentException("Invalid block value: " + value);
        }

        final var tag = matcher.group(GROUP_TAG);
        if (tag != null) {
            return blockTypeForTag(tag);
        }

        final var decimal = matcher.group(GROUP_DECIMAL);
        if (decimal != null) {
            try {
                return new BlockType(value, Long.parseLong(decimal, 10));
            } catch (NumberFormatException e) {
                throw new IllegalArgumentException("Decimal value out of range for block: " + value, e);
            }
        }

        var hash = matcher.group(GROUP_HASH);
        if (hash != null) {
            return new BlockType(hash, BLOCK_HASH_SENTINEL);
        }

        final var hexNum = matcher.group(GROUP_HEX_NUM);
        if (hexNum != null) {
            try {
                return new BlockType(hexNum, Long.parseLong(hexNum, 16));
            } catch (NumberFormatException e) {
                throw new IllegalArgumentException("Hex value out of range for block: " + value, e);
            }
        }
        throw new IllegalArgumentException("Invalid block value: " + value);
    }
