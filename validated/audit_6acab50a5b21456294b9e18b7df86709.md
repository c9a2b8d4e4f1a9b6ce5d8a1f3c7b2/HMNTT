Audit Report

## Title
Unauthenticated Mirror REST Response Allows Arbitrary gRPC Endpoint Injection via Unvalidated Node Network Map

## Summary
`fetchMirrorNodeNetwork` in `pinger/mirror_node_client.go` constructs the entire Hiero SDK consensus-node routing table from a plain-HTTP JSON response with no integrity verification, no allowlist of valid node account IDs, and no validation of endpoint addresses. An attacker able to intercept or spoof that HTTP response can inject arbitrary gRPC endpoints, causing every subsequent signed transaction to be routed to an attacker-controlled server.

## Finding Description
**`pinger/sdk_client.go` — `newClient()`, lines 17–22:**
When `cfg.network == "other"`, `buildNetworkFromMirrorNodes` is called and its result is passed directly to `hiero.ClientForNetwork(netmap)` with no further validation. [1](#0-0) 

**`pinger/mirror_node_client.go` — `fetchMirrorNodeNetwork()`, lines 74–131:**
The function issues a plain `http.Get`-equivalent request to `cfg.mirrorRest + "/api/v1/network/nodes"`, JSON-decodes the response body, and for every `nodeEntry` maps every `service_endpoints` entry directly into the network routing map. [2](#0-1) 

The only checks applied to each entry are:

1. **Empty-string guard** on `node_account_id` (line 103) — trivially satisfied by any non-empty string. [3](#0-2) 

2. **Syntactic parse** via `hiero.AccountIDFromString` (line 107) — accepts any well-formed `X.Y.Z` string; no allowlist of legitimate node account IDs (e.g., `0.0.3`–`0.0.28`) is enforced. [4](#0-3) 

3. **Non-empty host/port guard** (line 118) — accepts any non-empty host string and any non-zero port; no validation that the IP or domain belongs to a known consensus node. [5](#0-4) 

There is no TLS enforcement on the mirror REST connection, no certificate pinning, no HMAC/signature verification on the response, and no allowlist of expected node account IDs or endpoint addresses.

**`pinger/main.go` line 41:** `newClient` is called once at startup; a single intercepted response is sufficient to poison the entire lifetime of the client. [6](#0-5) 

## Impact Explanation
- **Transaction interception:** Every `cryptoTransfer.Execute(client)` call in `transfer.go` line 33 sends the fully-signed transaction to the attacker's gRPC endpoint, exposing transaction content (parties, amounts, transaction IDs, operator account). [7](#0-6) 
- **Selective denial of service:** The attacker can silently drop transactions or return gRPC errors; all retries (lines 23–57 of `transfer.go`) also route to the attacker. [8](#0-7) 
- **Fake receipt injection:** The attacker can return a crafted gRPC success response; `resp.GetReceipt(client)` would then query the (also attacker-controlled) mirror node for a receipt that does not exist on-chain, producing misleading monitoring data.
- **Severity: High** — full transaction routing hijack for the pinger's operator account for the entire process lifetime.

## Likelihood Explanation
The attack requires network-level access to intercept or spoof the HTTP response from the mirror REST endpoint. This is realistic in:
- Shared Kubernetes cluster networks (ARP spoofing, rogue pod on the same overlay network).
- Environments where the mirror node hostname resolves via mutable DNS (DNS poisoning, split-horizon DNS misconfiguration).
- Deployments where `HIERO_MIRROR_PINGER_REST` points to an external or semi-trusted HTTP endpoint.

The attack window is the single HTTP request made at startup (`main.go` line 41); no persistent access is required — one intercepted response is sufficient. [6](#0-5) 

## Recommendation
1. **Enforce HTTPS:** Reject any `cfg.mirrorRest` URL that does not use the `https` scheme, or configure the `http.Client` with a TLS transport that enforces certificate validation.
2. **Allowlist node account IDs:** After parsing, reject any `nodeAccountId` that does not appear in a configurable or hardcoded allowlist of expected consensus node account IDs for the target network.
3. **Validate endpoint addresses:** Optionally, validate that resolved IPs fall within expected CIDR ranges or that domain names match expected suffixes for the target network.
4. **Response integrity:** Consider adding an HMAC or signed-response mechanism to the mirror REST `/api/v1/network/nodes` endpoint, or cross-validate the fetched node list against a secondary trusted source before use.

## Proof of Concept
1. Position a listener to intercept the HTTP request to `cfg.mirrorRest` (e.g., via ARP spoofing or DNS poisoning of the mirror hostname in a shared Kubernetes namespace).
2. Return the following crafted JSON body:
```json
{
  "nodes": [
    {
      "node_account_id": "0.0.3",
      "service_endpoints": [
        { "ip_address_v4": "<attacker-ip>", "port": 50211 }
      ]
    }
  ],
  "links": { "next": null }
}
```
3. `fetchMirrorNodeNetwork` passes all three checks (non-empty account ID, valid `X.Y.Z` format, non-empty host and non-zero port) and returns `map["<attacker-ip>:50211"] = AccountID(0.0.3)`. [9](#0-8) 
4. `hiero.ClientForNetwork(netmap)` is called with this poisoned map; the SDK routes all consensus traffic to `<attacker-ip>:50211`. [10](#0-9) 
5. Every subsequent `cryptoTransfer.Execute(client)` sends the fully-signed transaction to the attacker's gRPC server. [11](#0-10)

```go
func fetchMirrorNodeNetwork(
	ctx context.Context,
	httpClient *http.Client,
	url string,
) (map[string]hiero.AccountID, bool, error) {
	req, err := http.NewRequestWithContext(ctx, http.MethodGet, url, nil)
	if err != nil {
		return nil, false, err
	}

	resp, err := httpClient.Do(req)
	if err != nil {
		return nil, true, fmt.Errorf("GET %s failed: %w", url, err)
	}
	defer resp.Body.Close()

	if resp.StatusCode < 200 || resp.StatusCode >= 300 {
		retry := resp.StatusCode == http.StatusTooManyRequests || resp.StatusCode >= 500
		return nil, retry, fmt.Errorf("GET %s returned %s", url, resp.Status)
	}

	var payload nodesEnvelope
	if err := json.NewDecoder(resp.Body).Decode(&payload); err != nil {
		return nil, false, fmt.Errorf("decode mirror nodes: %w", err)
	}

	network := make(map[string]hiero.AccountID)

	for _, n := range payload.Nodes {
		if n.NodeAccountID == "" {
			continue
		}

		nodeAccountId, err := hiero.AccountIDFromString(n.NodeAccountID)
		if err != nil {
			continue
		}

		// Use service_endpoints for node gRPC (what the SDK wants)
		for _, ep := range n.ServiceEndpoints {
			host := strings.TrimSpace(ep.DomainName)
			if host == "" {
				host = strings.TrimSpace(ep.IPAddressV4)
			}
			if host == "" || ep.Port == 0 {
				continue
			}

			addr := net.JoinHostPort(host, fmt.Sprintf("%d", ep.Port))
			network[addr] = nodeAccountId
		}
	}

	if len(network) == 0 {
		return nil, false, fmt.Errorf("no usable service_endpoints found from %s", url)
	}

	return network, false, nil
```
